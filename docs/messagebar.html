消息栏

Framework7 comes with special resizealbe toolbar for usage with Messages

消息栏的结构

消息栏的结构是很简单的:

<div class="toolbar messagebar" data-max-height="200">
  <div class="toolbar-inner">
    <textarea placeholder="Message"></textarea><a href="#" class="link">Send</a>
  </div>
</div>

Where

  • data-max-height - 限制Message Bar的最大高度

请注意它的位置，应该在”page“里边并且在"messages-content"前边:

<div class="page toolbar-fixed">
  <!-- messagebar -->
  <div class="toolbar messagebar">
    <div class="toolbar-inner">
      <textarea placeholder="Message"></textarea><a href="#" class="link">Send</a>
    </div>
  </div>
  <!-- messages-content -->
  <div class="page-content messages-content">
    <div class="messages">
      ... messages
    </div>
  </div>
</div>

相关的方法

myApp.destroyMessagebar(container) - 解绑所有的监听器和动作

  • container - HTMLElement or string (with CSS selector) - of message bar element (<div class="messagebar">) or of page that contains message bar

myApp.initMessagebar(container) - 在销毁之后绑定所有的事件和动作

  • container - HTMLElement or string (with CSS selector) - of message bar element (<div class="messagebar">) or of page that contains message bar
