幻灯片(Slider)

Framework7自带一个强大的幻灯片组件，集成了启用硬件加速的过渡动画效果和完美的仿native应用行为。它以著名的Swiper的方式内置在Framework7中。

幻灯片HTML布局结构

HTML结构相当简单：

<!-- Slider container -->
<div class="slider-container">
    <!-- Slider wrapper -->
    <div class="slider-wrapper">
        <!-- Slides -->
        <div class="slider-slide">Slide 1</div>
        <div class="slider-slide">Slide 2</div>
        <div class="slider-slide">Slide 3</div>
        ... other slides ...
    </div>
    <!-- 分页条，可选 -->
    <div class="slider-pagination"></div>
</div>

其中：

  • slider-container - 幻灯片和分页条的总容器。必需。

    • slider-wrapper - 附加的幻灯片容器（不包括分页条）。必需。

      • slider-slide - 单个幻灯片元素，可包含任何HTML结构。

    • slider-pagination -分页条的容器。注意，分页条会自动生成。可选。

通过JavaScript初始化幻灯片

现在我们有了幻灯片的HTML结构，还需要通过相关的App方法初始化它：

myApp.slider(sliderContainer, parameters) - 传入参数初始化幻灯片

  • sliderContainer - 幻灯片容器的DOM节点或CSS选择器。必需。
  • parameters - object - 幻灯片的对象参数。可选。
  • 方法返回初始化后的幻灯片实例

举个例子：

var mySlider = app.slider('.slider-container', {
    speed: 400,
    spaceBetween: 100
});
注意幻灯片容器在初始化时应该已经在DOM里存在了。所以如果你不是在首页使用幻灯片，应该在pageInit事件回调函数里执行初始化操作。

幻灯片参数

下面是所有可用的参数：

参数（Parameter） 类型（Type） 默认（Default） 描述（Description）
initialSlide number 0 初始展示的幻灯片序号
spaceBetween number 0 相邻幻灯片间距，单位像素（px）
slidesPerView number 1 同时可显示几张幻灯片（在幻灯片容器里同一时间可容纳展示的幻灯片数量）
direction string 'horizontal' 水平还是垂直滑动。可选值是'horizontal'和'vertical'
speed number 300 2张幻灯片间滑动过渡动画的持续时间，单位毫秒（ms）
preventClicks boolean true 设置为"true"可在滑动过程中阻止链接点击事件
preventClicksPropagation boolean true 设置为"true"可在滑动过程中阻止点击事件冒泡
loop boolean false 设置为 true，则循环播放
onlyExternal boolean false 设置为 true，则会禁用滑动和touch操作，因此你只能通过JS api比如.slideNext 或者 .slideTo来控制。
Pagination
pagination string / HTMLElement undefined 指定分页条的DOM节点或CSS选择器
paginationHide boolean true 当点击幻灯片容器时，分页条是否可见
自动播放
autoplay number - 自动播放模式下，2次滑动切换幻灯片动画间的时间间隙(ms)。如果这个参数没有指定，表示禁用自动播放
autoplayDisableOnInteraction boolean true 设置为false后，自动播放将不会因用户交互（比如手指拨弄滑动）而被停止，它会在每次用户交互后重新开始自动播放
导航按钮
nextButton string / HTMLElement undefined 传入DOM节点或CSS选择器，顾名思义，点一下往后翻一页
prevButton string / HTMLElement undefined 传入DOM节点或CSS选择器，顾名思义，点一下往前翻一页
indexButton string / HTMLElement undefined 传入DOM节点或CSS选择器，使分页条的每一项成为可点击的链接，点击哪一项则幻灯片滑动至对应页码的幻灯片显示
回调函数
onSlideChangeStart(slider) function 当幻灯片开始滑动动画（向前或后均可）时执行。接受一个幻灯片实例对象作为参数
onSlideChangeEnd(slider) function 当幻灯片结束滑动动画（向前或后均可）时执行。接受一个幻灯片实例对象作为参数
onTransitionStart(slider) function Callback function, will be executed in the beginning of transition. Receives slider instance as an argument.
onTransitionEnd(slider) function Callback function, will be executed after transition. Receives slider instance as an argument.
onTouchStart(slider, event) function 当用户开始触摸幻灯片时执行。接受一个幻灯片实例对象和'touchstart'事件作为参数
onTouchMove(slider, event) function 当用户按住幻灯片并移动手指时执行。接受一个幻灯片实例对象和'touchmove'事件作为参数
onOppositeTouchMove(slider, event) function 当用户按住幻灯片并以与"direction"参数值相反的方向移动手指时执行。接受一个幻灯片实例对象和'touchmove'事件作为参数
onSliderMove(slider, event) function 当幻灯片开始移动时执行，在onTouchMove之后触发。接受一个幻灯片实例对象和'touchmove'事件作为参数
onTouchEnd(slider, event) function 当用户结束触摸并释放手指时执行。接受一个幻灯片实例对象和'touchend'事件作为参数
onClick(slider, event) function 当用户点击/轻敲幻灯片区域300ms后执行。接受一个幻灯片实例对象和'touchend'事件作为参数
onTap(slider, event) function 当用户轻敲幻灯片区域时执行。接受一个幻灯片实例对象和'touchend'事件作为参数
onDoubleTap(slider, event) function 当用户连续2次轻敲幻灯片区域时执行。接受一个幻灯片实例对象和'touchend'事件作为参数
命名空间
slideClass string 'slider-slide' 幻灯片元素的CSS类名
slideActiveClass string 'slider-slide-active' 当前显示的幻灯片元素的CSS类名
slideNextClass string 'slider-slide-next' 紧邻当前显示的幻灯片下一张的CSS类名
slidePrevClass string 'slider-slide-prev' 紧邻当前显示的幻灯片上一张的CSS类名
wrapperClass string 'slider-wrapper' 幻灯片容器的CSS类名
bulletClass string 'slider-pagination-bullet' 分页条中每一项(表示每一页的那个点)的CSS类名
bulletActiveClass string 'slider-pagination-active' 分页条中当前显示项的CSS类名

幻灯片方法和属性

在我们初始化幻灯片后，我们拥有它的实例化对象(类似于上面的例子里的mySlider变量)，它本身带有一些方法和属性:

属性
mySlider.params 初始化时传入的参数对象
mySlider.container Dom7对象：幻灯片外层容器的HTML元素
mySlider.slides Dom7对象：每一张幻灯片的HTML元素集合
mySlider.activeSlideIndex 当前显示的幻灯片的序号
mySlider.previousSlideIndex 上一次显示的幻灯片的序号
mySlider.isFirst 若当前幻灯片在最左（或最上）的位置，为true
mySlider.isLast 若当前幻灯片在最右（或最下）的位置，为true
方法
mySlider.slideNext(); 滑动到下一张
mySlider.slidePrev(); 滑动到上一张
mySlider.slideTo(index, speed);
  • index - number - 目标页的页码序号
  • speed - number - 滑动动画持续时间
执行滑动动画到某张特定页码序号的幻灯片等效于'index'参数，持续时间等效于'speed'参数
mySlider.updateSlides(); 该方法会重新计算幻灯片数量和他们的位置。当你通过JS动态添加/删除了一些幻灯片时会有用
mySlider.updatePagination(); 重新计算 slider and slides sizes. 当你通过JS修改了幻灯片之后，可以通过此方法来更新。
mySlider.updateSize(); Dynamically creates pagination bullets
mySlider.updateClasses(); Updates active/prev/next classes on slides and bullets
mySlider.update(); This method contains .updateSlides, .updatePagination, .updateSize, .updateClasses. Should be called after you add/remove slides dynamically
mySlider.detachEvents(); 解除所有的事件监听器
mySlider.attachEvents(); 重新添加所有的事件监听器
mySlider.destroy(); 销毁幻灯片实例并解除所有事件监听器

通过HTML、无JS方式初始化

如果你不需要幻灯片组件的回调函数，那你可以通过HTML、无JS方式初始化。只需添加"slider-init"这个class到slider-container上

<!-- 添加“slider-init”来自动初始化幻灯片组件 -->
<div class="slider-container slider-init">
    <div class="slider-wrapper">
        <div class="slider-slide">Slide 1</div>
        <div class="slider-slide">Slide 2</div>
        <div class="slider-slide">Slide 3</div>
        ... other slides ...
    </div>
</div>

各种配置参数通过data-属性传入：

<!-- 在data-属性中传入幻灯片参数 -->
<div class="slider-container slider-init" data-speed="400" data-spaceBetween="40" data-pagination=".slider-pagination">
    <div class="slider-wrapper">
        <div class="slider-slide">Slide 1</div>
        <div class="slider-slide">Slide 2</div>
        <div class="slider-slide">Slide 3</div>
        ... other slides ...
    </div>
    <div class="slider-pagination"></div>
</div>

或者将所有参数以合法的JSON格式放在data-slider属性里：

<!-- 以JSON格式放在data-slider属性里的幻灯片参数 -->
<div class="slider-container slider-init" data-slider="{'speed':'400', 'spaceBetween':'40', 'pagination': '.slider-pagination'}">
    <div class="slider-wrapper">
        <div class="slider-slide">Slide 1</div>
        <div class="slider-slide">Slide 2</div>
        <div class="slider-slide">Slide 3</div>
        ... other slides ...
    </div>
    <div class="slider-pagination"></div>
</div>

访问幻灯片实例对象

即使你通过纯HTML方式初始化幻灯片组件，仍然可以访问其实例对象，它被挂载在幻灯片容器元素的'f7Slider'属性上：

var mySlider = $$('.slider-container')[0].f7Slider;
 
// 你可以用下面的方式调用幻灯片的方法
mySlider.slideNext();
