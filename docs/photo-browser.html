图片浏览器（Photo Browser）

Photo Browser 是一个像IOS图片浏览器一样用来展示图片的组件。 图片可以被缩放或者panned (可选)。

Photo Browser 使用 Slider 组件来展现图片。

创建一个图片浏览器实例

Photo Browser 只需通过javaScript就可以被创建和初始化。我们需要使用App封装好的方法:

myApp.photoBrowser(parameters) - 带参初始化 Photo Browser

  • parameters - object - Photo Browser 参数对象（必选）。
  • 该方法返回Photo Browser实例引用

案例:

var myPhotoBrowser = app.photoBrowser({
    zoom: 400,
    photos: ['path/to/image1.jpg', 'path/to/image2.jpg']
});   
myPhotoBrowser.open(); // open photo browser

Photo Browser 参数

所有可用参数列表：

Parameter Type Default Description
photos array [] 照片的url数组或者是包含“url”（或者“html”）和“caption”属性的对象数组.
initialSlide number 0 初始化时展示的图片序号
spaceBetween number 20 图片展现间隔距离（px）
speed number 300 切换图片的时间间隔（ms）
zoom boolean true 是否启用图片的缩放功能
maxZoom number 3 最大缩放指数
minZoom number 1 最小缩放指数
exposition boolean true 图片点击时是否启用浏览模式
expositionHideCaptions boolean false 是否隐藏在浏览模式中隐藏标题
swipeToClose boolean true 是否可以通过上下触摸的方式来关闭Photo Browser
view View instance undefined 如果想用page形式打开，则需要指定一个初始化的View实例，默认的如果不指定，将在Main View中打开
type string 'standalone' 定义Photo Browser的打开方式，可选参数有'standalone'（打开时将有一个自定义效果的遮罩层），'popup'（打开一个popup），’page‘（将被放置于View中，作为一个新页面加载）
theme string 'light' Photo Browser 颜色主题, 可以使用'light'或者'dark'
captionsTheme string 标题的颜色主题, 可以使用'light'或者'dark'。 默认的, 等于"theme"的参数
navbar boolean true 是否生成Photo Browser的导航栏
toolbar boolean true 是否生成Photo Browser的工具栏
backLinkText string 'Close' Photo Browser的导航栏中返回链接的文案
ofText string 'of' 图片的序号和总数之间的分割词，例如: "3 of 5"
模版
template string *Look below Photo Browser的html模版
navbarTemplate string *Look below 导航栏的html模版
toolbarTemplate string *Look below 工具栏的html模版
photoTemplate string *Look below 单独图片元素的html模版
objectTemplate string *Look below 单独对象元素的html模版
captionsTemplate string *Look below 标题容器的html模版
captionTemplate string *Look below 单独标题元素的html模版
回调
onOpen(photobrowser) function 回调函数，当Photo Browser被打开的时候执行
onClose(photobrowser) function 回调函数，当Photo Browser被关闭的时候执行
onSwipeToClose(photobrowser) function 回调函数，当Photo Browser通过上下触动的形式关闭时执行
幻灯片回调
onSlideChangeStart(slider) function 回调函数，当Photo Browser向前或者向后切换图片开始的时候执行，接收幻灯片的实例作为参数
onSlideChangeEnd(slider) function 回调函数，当Photo Browser向前或者向后切换图片结束的时候执行，接收幻灯片的实例作为参数
onClick(slider, event) function 回调函数，当Photo Browser被点击的时候延迟300ms执行，接收幻灯片的实例，touchend事件两个对象作为参数
onTap(slider, event) function 回调函数，当Photo Browser被触碰的时候执行，接收幻灯片的实例，touchend事件两个对象作为参数
onDoubleTap(slider, event) function 回调函数，当Photo Browser被连续两次触碰的时候执行，接收幻灯片的实例，touchend事件两个对象作为参数

Photo Browser 方法及属性

当我们初始化 Photo Browser 之后，我们将会得到它的实例引用(比如 myPhotoBrowser ) 并且包括很多有用的方法和属性:

属性
myPhotoBrowser.slider 包含初始化 Slider 的实例和所有可用的 Slider 方法和属性
myPhotoBrowser.container Dom7 形式的Photo Browser的html容器
myPhotoBrowser.exposed Photo Browser是否在浏览模式
myPhotoBrowser.activeSlideIndex 当前活动的幻灯片的序号
myPhotoBrowser.params 传入的初始化参数对象
方法
myPhotoBrowser.open(index); 根据’index‘参数的值打卡Photo Browser，如果’index‘参数没有被指定，则打开最后关闭的图片
myPhotoBrowser.close(); 关闭Photo Browser
myPhotoBrowser.toggleZoom(); 切换当前幻灯片的缩放形式
myPhotoBrowser.toggleExposition(); 切换浏览模式
myPhotoBrowser.expositionOn(); 启用浏览模式
myPhotoBrowser.expositionOff(); 禁用浏览模式

图片数组

当我们初始化Photo Browser的时候，需要传递一个数组，包括图片或者是视频的相关信息，让我们看看他们的区别吧:

// 如果我们需要标题的话，则需要包装成一个含有caption属性的对象:
var photos = [
    'image1.jpg',
    'image2.jpg',
    'image3.jpg',
    'image4.jpg',
];
 
//如果我们使用视频的时候，需要使用到html片段，那么就放到html属性中吧:
var photos = [
    {
        url: 'image1.jpg',
        caption: 'Caption 1'
    },
    {
        url: 'image2.jpg',
        caption: 'Caption 1'
    },
    // 这个是没有标题的
    {
        url: 'image3.jpg',
    },
    // 也可以不使用对象形式
    'image4.jpg'
];
 
//如果我们使用视频的时候，需要使用到html片段，那么就放到html属性中吧:
var photos = [
    {
        url: 'image1.jpg',
        caption: 'Caption 1'
    },
    // 带标题的视频
    {
        html: '<video src="video1.mp4"></video>',
        caption: 'Video Caption'
    },
    // 使用视频标签
    {
        html: '<iframe src="..."></iframe>',
    },
    // 这个是没有标题的视频
    '<video src="video2.mp4"></video>'
];

Photo Browser 模版

让我们来看看Photo Browser可用于初始化时的模版

Photo Browser 主模版

您可以通过使用模板参数定制的HTML模板。这是一个模板示例:

<div class="photo-browser photo-browser-dark">
    <div class="view navbar-fixed toolbar-fixed">
        {{navbar}}
        <div data-page="photo-browser-slides" class="page no-toolbar toolbar-fixed navbar-fixed">
            <div class="photo-browser-slider-container slider-container">
                <div class="photo-browser-slider-wrapper slider-wrapper">
                    {{photos}}
                </div>
            </div>
            {{toolbar}}
        </div>
    </div>
</div>

  • {{navbar}} - 会被导航栏模版所替换

  • {{toolbar}} - 会被工具栏模板替换

  • {{photos}} - 会被单独图片模板替换

导航栏模版

这是导航栏的示例模版，你可以通过 navbarTemplate 传入参数:

<div class="navbar">
    <div class="navbar-inner">
        <div class="left sliding">
            <a href="#" class="link close-popup photo-browser-close-link">
                <i class="icon icon-back"></i>
                <span>Close</span>
            </a>
        </div>
        <div class="center sliding">
            <span class="photo-browser-current"></span> <span class="photo-browser-of">of</span> <span class="photo-browser-total"></span>
        </div>
        <div class="right"></div>
    </div>
</div>

  • <a class="photo-browser-close-link"> - 当点击时将关闭 Photo Browser，并且解绑所有的监听器

  • <span class="photo-browser-current">. Photo Browser将插入当前的活动图片序号，并加载photo-browser-current 样式

  • <span class="photo-browser-total">. Photo Browser将插入幻灯片总数，并加载photo-browser-total 样式

工具栏模版

这是工具栏的示例模版，你可以通过toolbarTemplate 传入参数:

<div class="toolbar tabbar">
    <div class="toolbar-inner">
        <a href="#" class="link photo-browser-prev"><i class="icon icon-prev"></i></a>
        <a href="#" class="link photo-browser-next"><i class="icon icon-next"></i></a>
    </div>
</div>

这里是一个tab栏:

  • <a class="photo-browser-next"> - 使用 photo-browser-next 样式的下一个按钮

  • <a class="photo-browser-prev"> - 使用 photo-browser-prev 样式的上一个按钮

图片元素模版

这是一张幻灯片的模版，你可以通过 photoTemplate 传入参数:

<div class="photo-browser-slide slider-slide">
    <span class="photo-browser-zoom-container"><img src="{{url}}"></span>
</div>

对象元素模版

这是一个对象元素的模版，你可以通过 objectTemplate 传入参数:

<div class="photo-browser-slide photo-browser-object-slide slider-slide">
    {{html}}
</div>

标题模版

这是一个包含所有标题的模版，你可以通过 captionsTemplate 传入参数:

<div class="photo-browser-captions photo-browser-captions-dark">
    {{captions}}
</div>

单一标题模版

这是一个标题的模版，你可以通过 captionTemplate 传入参数:

<div class="photo-browser-caption" data-caption-index="{{captionIndex}}">
    {{caption}}
</div>
